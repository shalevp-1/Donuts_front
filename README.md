# Donuts Frontend

Frontend for **Donuts for You**, a full-stack e-commerce web application for browsing donuts, managing a cart, and completing purchases. Built with React and TypeScript, the app connects to a Node.js/Express backend and a MySQL database.

## Live Demo
Frontend: https://dountstime.netlify.app/ 

## Features
- Browse a live donut catalog loaded from the backend
- View product details including price, ingredients, and calories
- Add and remove items from a persistent cart
- Register, log in, and log out securely
- Complete checkout and receive an order summary
- Access a member page with rewards points, coupons, and purchase history
- Access admin-only pages to manage donuts and users
